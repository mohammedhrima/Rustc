## Controls
|Keys|Description|
|---|---|
|<kbd>keyUp</kbd>, <kbd>keyDown</kbd>| Move up and Move down|
|<kbd>q</kbd>|Quit|
|<kbd>Enter</kbd>| Perform action on the highlighted element
|<kbd>TAB</kbd>| to switch between TODO and DONE
|<kbd>s</kbd>| to save
